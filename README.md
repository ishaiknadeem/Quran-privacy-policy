# Privacy Policy for Quran Meanings & Audio

Last Updated: March 2026

Quran Meanings & Audio (“the App”) is committed to protecting your privacy. This Privacy Policy explains how we handle information when you use the App. Our approach is built on minimizing data collection and ensuring a secure, private user experience.



 1. Data Collection

We do not collect, store, or share any personal or sensitive user data.

This includes (but is not limited to):

* Name, email address, or phone number
* Location information
* Device identifiers (such as device ID or IP address)
* User behavior, analytics, or tracking data

The App is designed to function without requiring any personal information.



 2. Local Storage

The App stores certain data locally on your device to enhance user experience. This data never leaves your device and is not transmitted to any external servers.

This includes:

* Bookmarks – verses saved for quick access
* Reading Progress – last read Surah or Ayah
* Preferences – settings such as font size and audio configuration



 3. Third-Party Services

To deliver core functionality, the App uses trusted external services to fetch Quranic content and audio:

* Al Quran Cloud API – provides Quran text and translations
* EveryAyah CDN – streams audio recitations

These services receive only technical requests (e.g., Surah and Ayah numbers).
No personal or identifiable user data is shared with these services.



 4. Permissions

The App may request the following permissions:

* Internet Access – required to load Quran text and stream audio
* Vibration Control – used for optional haptic feedback

These permissions are used solely for app functionality and not for data collection.



 5. Children’s Privacy

The App does not collect any personal data and is safe for users of all ages, including children under 13.



 6. Updates to This Policy

This Privacy Policy may be updated periodically. Any changes will be reflected within the App or on the official policy page.
